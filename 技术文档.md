# A股量化分析平台 — 技术文档

## 一、数据来源

| 数据 | 接口 | 说明 |
|------|------|------|
| 日K线（OHLCV） | `web.ifzq.gtimg.cn` 腾讯财经 | 前复权日线数据，包含开盘/收盘/最高/最低/成交量 |
| 股票列表（代码+名称） | `akshare.stock_info_a_code_name()` | 全A股代码名称表，用于搜索匹配 |
| 实时行情（备用） | `akshare.stock_zh_a_spot_em()` 东方财富 | 含PE/PB/市值/换手率，网络不稳定时回退到历史数据 |
| 指数K线 | `web.ifzq.gtimg.cn` 腾讯财经 | 上证指数(sh000001)日线，用于计算Beta |
| 指数估值 | `akshare.stock_index_pe_lg()` / `stock_index_pb_lg()` | 沪深300 PE/PB 历史分位，作为估值参考 |

> 所有数据均免费，无需 API Key。

## 二、技术指标

### 2.1 移动平均线 (MA)

| 指标 | 周期 | 计算方式 |
|------|------|---------|
| MA5 | 5日 | `close.rolling(5).mean()` |
| MA10 | 10日 | `close.rolling(10).mean()` |
| MA20 | 20日 | `close.rolling(20).mean()` |
| MA60 | 60日 | `close.rolling(60).mean()` |
| MA120 | 120日 | `close.rolling(120).mean()` |
| MA250 | 250日 | `close.rolling(250).mean()` |

**信号判断：**
- 多头排列：MA5 > MA20 > MA60 → 强势
- 空头排列：MA5 < MA20 < MA60 → 弱势
- 金叉/死叉：MA5 与 MA20 交叉

### 2.2 MACD

| 参数 | 值 | 公式 |
|------|-----|------|
| 快线 | 12 | `EMA(close, 12)` |
| 慢线 | 26 | `EMA(close, 26)` |
| DIF | — | `EMA12 - EMA26` |
| DEA | 9 | `EMA(DIF, 9)` |
| 柱线 | — | `2 × (DIF - DEA)` |

**信号判断：**
- 金叉：DIF 上穿 DEA → 看涨
- 死叉：DIF 下穿 DEA → 看跌
- 红柱放大 → 多头增强；绿柱放大 → 空头增强

### 2.3 RSI（相对强弱指标）

| 参数 | 值 |
|------|-----|
| 周期 | 14日 |
| 平滑方式 | EMA（Wilder's smoothing） |

**信号判断：**
- RSI > 80：严重超买
- RSI > 70：超买区域
- RSI < 20：严重超卖
- RSI < 30：超卖区域
- 40~60：中性区域

### 2.4 KDJ（随机指标）

| 参数 | 值 |
|------|-----|
| K周期 | 9日 |
| K平滑 | 3日EMA |
| D平滑 | 3日EMA |
| J值 | `3K - 2D` |

**信号判断：**
- K/D < 20：超卖区域
- K/D > 80：超买区域
- K > D：多头
- K < D：空头

### 2.5 布林带 (Bollinger Bands)

| 参数 | 值 | 公式 |
|------|-----|------|
| 中轨 | 20日均线 | `MA(close, 20)` |
| 上轨 | ±2σ | `中轨 + 2 × std(20)` |
| 下轨 | ±2σ | `中轨 - 2 × std(20)` |

**信号判断：**
- 价格触及上轨 → 压力位
- 价格触及下轨 → 支撑位
- 带宽收窄 → 变盘预警（突破即将发生）

### 2.6 ATR（平均真实波幅）

| 参数 | 值 | 公式 |
|------|-----|------|
| 周期 | 14日 | `EMA(max(H-L, |H-C₁|, |L-C₁|), 14)` |

用途：止盈止损计算参考。建议止损 = 当前价 − 2×ATR，建议止盈 = 当前价 + 3×ATR。

### 2.7 成交量均线

| 指标 | 周期 |
|------|------|
| VOL_MA5 | 5日均量 |
| VOL_MA20 | 20日均量 |

## 三、估值分析

| 指标 | 数据来源 | 说明 |
|------|---------|------|
| PE_TTM | 东方财富 spot（备用：历史数据不可用） | 滚动市盈率 |
| PB | 东方财富 spot（备用：历史数据不可用） | 市净率 |
| PE历史分位 | akshare 财报数据 | 近5年PE在历史中的位置，越低越便宜 |
| PB历史分位 | akshare 财报数据 | 近5年PB在历史中的位置 |
| 沪深300 PE/PB | akshare `stock_index_pe_lg/pb_lg` | 大盘估值参考 |

**估值评级规则：**

| 分位区间 | 评级 |
|---------|------|
| < 10% | 显著低估 |
| 10%~25% | 估值偏低 |
| 25%~75% | 估值合理 |
| 75%~90% | 估值偏高 |
| > 90% | 显著高估 |

## 四、风险指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **VaR(95%)** | 日收益率第5百分位数 | 95%概率下每日最大亏损 |
| **VaR(99%)** | 日收益率第1百分位数 | 99%概率下每日最大亏损 |
| **CVaR(95%)** | 低于VaR(95%)的收益率均值 | 尾部极端风险 |
| **Sharpe Ratio** | `(年化收益 - 无风险利率) / 年化波动` | 风险调整后收益，>1良好，>2优秀 |
| **最大回撤** | `max((累计净值 - 历史最高) / 历史最高)` | 历史最大亏损幅度 |
| **回撤天数** | 最长持续回撤期 | 净值在水下最长天数 |
| **年化波动率** | `日收益std × √252` | 价格波动程度 |
| **年化收益率** | `日均收益 × 252` | 年化后的平均收益 |
| **Calmar Ratio** | `年化收益 / 最大回撤` | 每单位最大风险带来的回报 |
| **Beta** | `Cov(个股, 上证) / Var(上证)` | 相对大盘的敏感度 |
| **日胜率** | 上涨天数 / 总交易天数 | 每日赚钱概率 |

## 五、回测策略

### 5.1 策略一览

| 策略ID | 名称 | 类型 | 核心逻辑 |
|--------|------|------|---------|
| `ma_cross` | MA双均线交叉 | 趋势跟踪 | MA快线(5)上穿MA慢线(20)→买入；下穿→卖出 |
| `macd_signal` | MACD信号 | 动量 | DIF上穿DEA→买入；下穿→卖出 |
| `rsi_mean_rev` | RSI均值回归 | 反转 | RSI从超卖区(30)回升→买入；从超买区(70)回落→卖出 |
| `boll_break` | 布林带突破 | 波动突破 | 收盘价突破上轨→买入；跌破中轨→卖出 |
| `boll_mean_rev` | 布林带回归 | 均值回归 | 收盘价跌破下轨→买入；回升至中轨→卖出 |
| `triple_screen` | 三重滤网 | 多周期确认 | 潮汐(MA多头排列) + 波浪(MACD金叉) + 浪花(RSI不超买) 三重确认 |
| `turtle_channel` | 海龟通道 | 趋势突破 | 突破N日(20)最高价→买入；跌破M日(10)最低价→卖出 |

### 5.2 回测参数设定

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 初始资金 | ¥100,000 | 可自定义 |
| 手续费 | 0.03%/单边 | 含佣金+印花税 |
| 滑点 | 0.1% | 模拟成交价偏差 |
| 仓位模式 | 全仓进出 | 要么满仓持股，要么空仓现金 |
| 做空 | 不支持 | 符合A股散户实际 |
| 执行延迟 | T日信号→T日收盘执行 | 避免前视偏差 |

### 5.3 回测绩效指标

| 指标 | 说明 |
|------|------|
| 总收益 | 期末权益 vs 初始资金 |
| 年化收益 | 按252个交易日年化 |
| 基准收益 | 同期 buy-and-hold 策略收益 |
| 超额收益 | 策略收益 − 基准收益 |
| Sharpe Ratio | 风险调整收益 |
| 最大回撤 | 最大净值回撤幅度 |
| 胜率 | 盈利交易占比 |
| 盈亏比 | 总盈利 ÷ 总亏损 |
| 平均盈利/亏损 | 单笔平均盈亏 |
| 交易次数 | 总交易笔数 |
| 最大连续亏损 | 最长连亏次数 |
| 期望值 | `胜率×平均盈利 − (1−胜率)×平均亏损` |

## 六、操作建议引擎

### 6.1 四维度加权评分（满分100）

| 维度 | 满分 | 评估内容 |
|------|------|---------|
| **趋势** | 25分 | 均线排列方向、MACD金叉/死叉 |
| **动量** | 25分 | RSI超买超卖、KDJ方向、布林带位置 |
| **估值** | 25分 | PE/PB分位水平、估值评级 |
| **风险** | 25分 | Sharpe、最大回撤、波动率、Beta |

### 6.2 信号判定

| 总分区间 | 信号 | 置信度 |
|---------|------|--------|
| ≥ 75 | 🟢 买入 | 高 |
| 60 ~ 74 | 🟢 买入 | 中 |
| 45 ~ 59 | 🟡 持有 | 中 |
| 30 ~ 44 | 🟡 持有 | 低 |
| < 30 | 🔴 卖出 | 中（<20）/ 低（20-29） |

### 6.3 仓位计算

```
建议仓位 = (总分/100) × 25% × (25 / max(波动率, 10))
```

- 下限 5%，上限 25%
- 波动率越低 → 仓位越高

### 6.4 止盈止损

- 止损价 = 当前价 − 2 × ATR(14)，或 当前价 × 0.92（取较保守者）
- 止盈价 = 当前价 + 3 × ATR(14)，或 当前价 × 1.20

## 八、风险提示

> ⚠️ 本工具仅供学习研究参考，不构成投资建议。
>
> - 回测收益不代表未来表现，策略参数可能过拟合历史数据
> - 技术指标存在滞后性，信号出现时行情可能已变化
> - PE/PB 数据依赖实时接口，网络不稳定时可能缺失
> - 评分和建议来自规则引擎，无法预测黑天鹅事件
> - 实际交易需考虑流动性、滑点、情绪等因素
